## About This Repository

This repository contains documented malware samples designed to help security professionals, researchers, and students understand:
- Modern malware techniques and tactics
- Command & Control (C2) infrastructure
- Evasion and obfuscation techniques
- Detection and mitigation strategies

Each project includes comprehensive documentation, source code, and analysis to facilitate learning and improve defensive capabilities.

## Getting Started

1. **Read the Disclaimer**: Ensure you understand the legal and ethical implications
2. **Browse the Catalog**: Navigate to [samples/README.md](samples/README.md) to see all available samples
3. **Study the Documentation**: Each project has detailed documentation explaining techniques and implementations
4. **Set Up Lab Environment**: Always use isolated VMs or containers for testing
5. **Learn Defensively**: Focus on understanding how to detect and prevent these techniques
